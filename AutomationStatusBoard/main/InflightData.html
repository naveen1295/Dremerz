Test Data Creation

The script can create Quote/Application with data options for Finance type, Borrower Type, GFV, Application Status, Campaign and if you want to use random data or provide own data in the excel for Individual. Below are the detail steps eplaining how to create the excel sheet with inflight data that you need to provide to Automation Team.
Click here to download the sample inflight excel sheet and follow below instructions in filling the excel.

  • Do not add/delete any columns in the excel. Do not rename the excel or excel sheet names
  • Each row in the excel represent either a Quote/Application to be created
  • Mandatory Fields: These columns in the excel need to have data
    • columns E,F,G,H,J,K,L
    • Please select the values from the dropdown, do not copy paste or type any data into these excel cells
    • If you want a new row, copy paste the existing row and select values from drop down for above columns
    • Column G - Status
      1. 'Blank' with Quote combination means create Quote till loan details page
      2. 'Blank' with application combination means create quote and convert it into appliation on borrower screen
      3. Rest of the status are self explanatory in the excel
  • Optional Fields:
    • Column I - Campaign, Make sure there are no spelling mistakes or extra spaces
    • Below columns in the excel need to have data only if column L has "Yes"
  • Always copy data to notepad and then to excel, this will eliminate any special characters copied
