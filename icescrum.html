    Hosting icescrum on local

    • Downlaod the icescrum jar file
    • Open command prompt and go the downloaded jar file location
    • Launch iceScrum from the command line using: "java -Xmx1024M -XX:MaxPermSize=256m -jar icescrum.jar"
    • Open your browser and go to suggested URL from the command line (usually: http://localhost:8080/icescrum/)
