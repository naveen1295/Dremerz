SQL (Structured Query Language)

Database is a collection of data stored in a format that be easily accessed. Database Management System(DBMS) is used to manage the database. DBMS is classified into 2 categories Relational and NoSQL RDBMS - Data is stored in tables and these tables are linked to each other. SQL is used to query/modify the Relational Database. Major RDBMS are MySQL,SQL Server,Oracle. No SQL - doesn't have tables or relations, they have thier own query Language(not SQL).
